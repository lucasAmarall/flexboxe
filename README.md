# flexboxe


A TypeScript Flexbox builder with various properties and methods to build and manipulate Flexbox layouts.

## Props

```
{
  flex:(),
  order:(),
  shrink:(),
  basis:(),
  grow:(),
  justifyContent: {
    start:(),
    end:(),
    center:(),
    stretch:(),
    between:(),
    around:()
  },
  alignContent: {
    start:(),
    end:(),
    center:(),
    stretch:(),
    between:(),
    around:()
  },
  alignItems: {
    start:(),
    end:(),
    center:(),
    stretch:(),
    between:(),
    around:()
  },
  flexDirection: {
    row:(),
    column:(),
    columnReverse:(),
    rowReverse:()
  },
  margin: {
    top:(),
    left:(),
    right:(),
    bottom:(),
    vertical:(),
    horizontal:()
  },
  padding: {
    top:(),
    left:(),
    right:(),
    bottom:(),
    vertical:(),
    horizontal:()
  },
  wrap: {
    nowrap:(),
    wrap:(),
    wrapReverse:()
  },
  build:()
}
```
